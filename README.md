# Tower of Blocks

Мини-игра по ТЗ: горизонтальный скролл кубиков внизу, перенос в башню справа вверху, выкидывание в яму слева вверху.

---

## Анимации (требование ТЗ)

В ТЗ указано: *«Должна использоваться как минимум 1 анимация (Unity Animation или DoTween)»*.

В проекте вместо DoTween используется **LitMotion** ([Annulus/LitMotion](https://github.com/AnnulusGames/LitMotion)) — современная tween-библиотека для Unity с нулевым аллокациями и поддержкой async. Все игровые анимации реализованы через LitMotion:

- **PlaceBounce** — подскок при постановке блока на башню (вверх → вниз)
- **HoleFall** — дуга и затухание при сбросе блока в яму
- **CollapseDown** — плавное опускание блоков после удаления из башни
- **MissDisappear** — исчезновение (fade-out) при промахе

Использование LitMotion эквивалентно требованию «как минимум 1 анимация» и покрывает все сценарии движения и исчезновения блоков.
